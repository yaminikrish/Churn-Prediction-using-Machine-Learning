About the Machine Learning Model

The Customer Churn Prediction Model is built to help telecom companies predict customers who are likely to discontinue their services. It analyzes various demographic, service, and billing-related features to make accurate churn predictions.

Model Workflow

  • Performed data preprocessing including missing value handling and standardization using StandardScaler.
  • Encoded categorical variables using One-Hot Encoding for model interpretability.
  • Handled outliers using IQR capping and balanced data using SMOTE technique.
  • Predicted churn as binary output — 0: Customer stays, 1: Customer churns.

Features Used

  • Demographics: Gender, SeniorCitizen, Partner, Dependents
  • Services: PhoneService, InternetService, OnlineBackup, TechSupport, StreamingTV
  • Billing & Contract: PaymentMethod, Contract, PaperlessBilling
  • Numerical: MonthlyCharges, TotalCharges, Tenure

Model Finalization

After studying and experimenting with multiple machine learning algorithms like Logistic Regression, Decision Tree, and Random Forest, the Gradient Boosting Classifier was finalized for its superior accuracy, robustness, and ability to handle complex feature interactions effectively.

Purpose

This model empowers telecom providers to predict customer churn in advance, enabling them to design targeted retention strategies and improve long-term customer loyalty.
